# Room Maintenance Workflow

## Purpose

Room Maintenance is for room setup and room record correction, not daily front-desk booking work.

## Intended usage

1. Open Room from `F4` or navigation.
2. Select a room from the room list.
3. Review room details on the right:
   - room number
   - description
   - type
   - location
   - rate
   - breakfast settings
   - maintenance and housekeeping state
   - active flag
4. Save changes.

## Use this screen when

- a room needs initial setup
- room rate or type changed
- room location or description is wrong
- breakfast configuration changed
- a room should be marked active or inactive
- room metadata needs review

## Avoid using this screen when

- the goal is to check in or check out a guest
- the operator only needs to open the current booking
- the room state should be changed as part of a live arrival or departure transaction

## Intended mental model

Room Maintenance owns room master data. The Room Board owns live operational flow.
